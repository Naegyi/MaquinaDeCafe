Máquina de Café

Um programa que simula o funcionamento de uma máquina de café.

O terminal apresenta comandos para o cliente escolher sua bebida. Além disso, existem comandos especiais conhecidos apenas pelo funcionário:

    reporte – exibe o status atual dos ingredientes e do dinheiro no sistema.

    reestoque – permite reabastecer os ingredientes da máquina.

Para desligar a máquina, quando o programa perguntar "O que você quer?", basta digitar off e o código será encerrado.
