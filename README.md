# MaquinaDeCafe
Um código que simula uma maquina de café


Nesse programa, além dos comando mostrados pelo terminal temod mais alguns que seriam de conhecimento só do funcionario como:

1 - reporte
2 - reestoque

No caso de querer desligar a maquina, quando ele perguntar oque você quer poder escrever 'off' e o código para.
